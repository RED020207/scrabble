# SCRABBLE
*This is a small side-project I made while learning full-stack web development.*

It uses **CSS**, **HTML** & **JavaScript**.

---

To put it simply, this is a word puzzle game that heavily tests one's vocabulary. You are provided with a *scrambled* mix of letters, and you must click them in the order that makes them form a *meaningful* word. ***Everything is randomized.***

There are **5 levels of difficulty** available:
- **Easy**
- **Medium**
- **Hard**
- **Very Hard**
- **Insane**

There's also a **Hints** system available, usable on *all levels **except Easy***. It rearranges the first half of the word in correct order, leaving you to handle the rest.

*High scores for each level are stored **locally**, allowing replayability and competing with oneself to score higher than before.*

[**NOTE**: Sometimes loading may take a while, since its using a free community-driven random word generator API that can have occasional downtimes. Local arrays of relevant words for the difficulty levels are also thus present for uninterrupted gameplay.]
