# SCRABBLE
*This is a fun little side-project I made while learning full-stack web development.*

It uses **CSS**, **HTML** & **JavaScript**.

---

Now, to put it simply, this is a word puzzle game that heavily tests one's vocabulary. You are provided with a *scrambled* mix of letters, and you must click them in the order that makes them form a *meaningful* word. ***Everything is randomized.***

There are **5 levels of difficulty** available:
- **Easy**:   *3-4* letter words
- **Medium**: *5-6* letter words
- **Hard**: *7-8* letter words
- **Very Hard**: *9-10* letter words
- **Insane**: *13-14* letter words

There's also a **Hints** system available, usable on *all levels **except Easy***. It rearranges the first half of all the letters in correct order, leaving you to handle the rest.

*High scores for each level are stored **locally**, allowing replayability and competing with oneself to score higher than before.*
